This is basic user authentication application which contains express module with the complete implementation of sessions  created so that
you can use this aand startworking on your nodeJs application directly using the code . This code create user session as you login . 
You can store the users in the date by storing the user array in app.js file 

-> Download the Nodejs 
-> Open folder 'NodeJS-Passport-Login' in your editor or IDE
-> Fire up the terminal with :
               npm install express
               npm install express-session
	       npm install express-flash
               npm install method-override
               npm install cookie-parser
               npm install uuid
               npm install bcrypt
               npm install passport

->Finally run command :
               node app.js
->open 'localhost:3000/' on  browser
